6. 네이밍

클래스의 경우
첫 문자는 대문자로 시작합니다.
ex) MyHomeTown
클래스 안의 자식 클래스 일 경우는 부모클래스네이밍_(언더바)자식 클래스네이밍 을 사용해 구분합니다.
ex) 부모 클래스 = "ParentClass" 자식 클래스 = "ChildClass" 
=> 클래스 네이밍 = "ParentClass_ChildClass" 가 됩니다.

함수나 메서드의 경우
첫글자는 소문자 뒷글자는 대문자로 시작합니다.
ex) function myHome( .. )

지역변수의 경우
소문자로 시작하면 2개의 문자를 혼합해 사용할 경우 문자 사이에 _(언더바) 를 두어 문자_문자로 표기한다.
ex) $get_data = "1";

상수의 경우
항상 대문자로 시작하며 단락은 _(언더바)를 사용한다.
ex) define(MAX_DATA,true);

전역변수의 경우
지역변수와 구분을 위해 _(언더바)로 시작합니다.
ex) $_SESSION, $_SERVER 등과 같이 표기한다.
